# Candy Crush Game (C# - Windows Forms)

## Project Overview

This project is a Candy Crush-style puzzle game developed in C# using Windows Forms. The goal of the project is to implement grid-based game mechanics while applying object-oriented programming principles.

The game allows users to match elements on a grid, calculate scores, and manage game states dynamically.

## Purpose

- Practice object-oriented programming (OOP)
- Implement grid-based game logic
- Design an interactive desktop application
- Apply algorithmic thinking in game mechanics

## Features

- Grid-based game board
- Matching system for game elements
- Score calculation
- Game state management
- Interactive user interface (Windows Forms)
- Basic game loop mechanics

## Technologies Used

- C#
- Windows Forms
- Object-Oriented Programming (OOP)

## Game Logic

- The game is based on a 2D grid system
- Matching similar elements removes them from the board
- The grid updates dynamically after each move
- Scores are calculated based on successful matches
- Game state is continuously managed during gameplay
